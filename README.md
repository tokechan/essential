# Node.js Webアプリケーション開発

## 概要
このプロジェクトは、Node.jsとExpressフレームワークを使用したWebアプリケーション開発の学習用リポジトリです。

## プロジェクト構成
- `/sample` - Node.jsの基本操作（ファイル操作、JSON処理など）の学習用コード
- `/web-server` - Express.jsを使用したWebサーバーの実装（予定）

## 学習内容
### 完了
- Node.jsの基本概念
- ファイル操作（fs モジュール）
- JSONデータの取り扱い
- モジュールのエクスポート/インポート
- 非同期処理の基礎

### 予定
- Express.jsの基本設定
- ルーティング
- ミドルウェアの使用
- RESTful APIの実装
- データベース連携

## 環境構築
```bash
npm install
```

## 使用技術
- Node.js
- Express.js
- その他必要なパッケージ（随時追加予定）
